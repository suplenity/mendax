## **Mendax: discord quest completor**

#### Mendax is a quick project that i worked on for maybe a week, it works both on modded discord clients (betterdiscord, vencord or equicord) and **vanilla** clients aswell by remote controlling the app from the outside

### **Features:**
1. Auto complete any quests, supported quests:
- Video quests
- game quests
- stream quests (if they even exist anymore???)
- console quests
- activity quests (instantly)

2. Quest queue - line up multiple quests and run them back to back in the background (yes i added a "send to system tray" button aswell)

3. sorting quests - so u can pick the one that will expire soon, or just by what u want as a reward

4. custom finished notification - so when its done, it lets u know!

# !! WARNING !!
This does not fake a running exe with a custom name, this uses the same method that discord detected a while back and sent warning messages to users to stop doing this, u can use mendax, **but ur risking a warning**
